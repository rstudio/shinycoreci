# Assert [`test_in_local()`](https://rstudio.github.io/shinycoreci/reference/test_in_local.md) output

Method called when
[`test_in_local()`](https://rstudio.github.io/shinycoreci/reference/test_in_local.md)
is called with `assert = TRUE`.

## Usage

``` r
assert_test_output(output)
```

## Arguments

- output:

  value received from
  [`test_in_local()`](https://rstudio.github.io/shinycoreci/reference/test_in_local.md)
