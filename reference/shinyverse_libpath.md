# Shinyverse libpath

Methods to get and reset the shinyverse `libpath`.

## Usage

``` r
shinyverse_libpath()

shinyverse_clean_libpath()
```

## Functions

- `shinyverse_libpath()`: Library path that will persist across
  installations. But will have a different path for different R versions

- `shinyverse_clean_libpath()`: Removes the cached R library
