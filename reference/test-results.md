# View (and save) `test_in_local()` output

Use `view_test_results()` to preview test results generated by
`save_test_results()` (saving should only ever be done in a GHA
workflow).

## Usage

``` r
save_test_results(
  output,
  gha_branch_name,
  pr_number,
  username,
  repo_dir = rprojroot::find_package_root_file()
)
```

## Arguments

- output:

  output from
  [`test_in_local()`](https://rstudio.github.io/shinycoreci/reference/test_in_local.md).

- gha_branch_name:

  an identifier determined by the GHA workflow

- pr_number:

  the pull request number.

- username:

  the username of the github profile.

- repo_dir:

  Location of local shinycoreci repo
