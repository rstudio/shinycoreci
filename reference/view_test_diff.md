# View Shinytest Diff

View Shinytest Diff

## Usage

``` r
view_test_diff(suffix = platform_rversion(), dir = "apps", ...)
```

## Arguments

- suffix:

  Test output suffix to compare against

- dir:

  Root folder path

- ...:

  Extra arguments passed to
  [`shinytest::viewTestDiff`](https://rdrr.io/pkg/shinytest/man/viewTestDiff.html)
